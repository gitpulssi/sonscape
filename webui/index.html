Enable this mode to receive 8-channel audio from an external computer over WiFi. The external computer must send audio at 48kHz, 1200 samples per block.

Connection Info

Sample Rate: 48000 Hz

⚠️ Important Notes

  • This mode disables built-in therapy signal generation
  • Bluetooth audio mixing still works
  • External computer must send data continuously to avoid audio gaps
  • Network latency affects audio quality - use wired connection for best results
  • Example Python client code available in ws_audio.py comments
